### 사용설명


>2.1과 2.2 파일 모두 ipynb환경에서

RunAll 버튼 한번 누르면 자동으로 파일생성됩니다.

디폴트로 사용자 입력값 받게 설정해놓았고,
옵션(선택사항)으로 직접 변수에 접근하여 문자열 변경가능합니다.

사용자 입력시 대륙,국가 모두(한/영) 입력 가능합니다.
다만 현재 국가명은 아시아국가만 구현되어 있어 추후 필요할시 업데이트 하겠습니다.


#### 알림 : 
2.2ipynb 파일의 경우 Plotly 그래프를 포함하므로
추가 Dependency 설치 해야 Plotly그래프가 정상 출력 됩니다.

Dependency목록 :
- pip install plotly
- pip install nbformat>=4.2.0
- pip install kaleido

#### 그래프폴더 위치 변경란 :

> 기본적으로 그래프폴더위치는 아직 손볼곳이 많이 있습니다.
그래프 저장시 원하시는 위치에 직접 폴더패스입력 하시거나 변수 만들어서 
자유롭게 변경하시면 될것같습니다.
